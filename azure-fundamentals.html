About This Course

This course prepares you for the Microsoft Azure Fundamentals (AZ‑900) certification exam. You will learn cloud concepts, Azure architecture, security, governance, and essential Azure services.

What You Will Learn

  • • Cloud concepts and shared responsibility model
  • • Azure global infrastructure
  • • Compute, networking, storage, and databases
  • • Azure security, identity, and governance
  • • Pricing, SLAs, and lifecycle
  • • Exam preparation and practice questions

Course Modules

  • Module 1: Cloud Concepts
  • Module 2: Azure Architecture & Services
  • Module 3: Compute, Storage & Networking
  • Module 4: Security, Identity & Governance
  • Module 5: Pricing & Support
  • Module 6: Exam Preparation

Exam Information

  • • Exam code: AZ‑900
  • • Duration: 60 minutes
  • • Format: Multiple choice
  • • Delivery: Online or test center
